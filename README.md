# Calton Ates — Portfolio Site

Personal portfolio site built as a single-page HTML/CSS/JS site. No frameworks or build tools — just static files served via GitHub Pages.

## Stack

- HTML, CSS, JavaScript (vanilla)
- [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) + [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- CSS custom properties for theming, `clamp()` for fluid type, CSS Grid for layout
- `IntersectionObserver` for scroll fade-ins
